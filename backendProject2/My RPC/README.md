# remote-procedure-call

## 概要
node.js 側のクライアントが入力したデータを，Pythonで記述されたサーバが処理を行うJSON-RPCです.

## 実行方法
サーバー側
```
python3 rpc-server.py
```
クライアント側
```
node rpc-client.js
```
